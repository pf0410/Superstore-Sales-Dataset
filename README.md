Superstore Sales Dataset
零售绩效与盈利能力诊断仪表盘

本项目基于 Superstore Sales Dataset，通过数据分析方法对零售业务的销售表现与盈利结构进行探索性分析，并构建数据可视化仪表盘。

数据集来源：
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting/versions/1/data

项目概述
本项目使用 Python 对数据进行基础清洗与预处理，并利用 MySQL 进行数据存储与中转，最终通过 Power BI 构建数据分析仪表盘。

分析重点包括：
企业销售收入变化趋势
产品类别盈利结构
客户贡献度分布
产品销售效率

不同于简单的数据可视化，本项目更关注通过数据分析 识别潜在业务风险、发现业务增长驱动因素，并揭示可能存在的结构性盈利问题。

技术实现
数据清洗：Python（Pandas）
数据存储：MySQL
数据可视化：Power BI

关键分析发现 (Key Findings)
1. 明显的季节性销售模式
    销售收入在 第四季度（Q4）持续达到峰值，说明企业销售具有明显的节假日驱动需求特征。
2. 科技类产品是主要增长引擎
    Technology（科技类） 产品类别贡献了 最高销售额且利润率较高，是推动公司收入增长的重要业务板块。
3. 存在结构性盈利问题
    部分子类别（如 Tables）虽然销售额较高，但 利润为负，说明可能存在 定价策略不合理或成本控制问题。
4. 收入集中度风险
    少数头部客户贡献了较大比例的销售收入，企业收入结构存在一定的客户集中度风险。
