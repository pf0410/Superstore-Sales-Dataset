SuperstoreSalesDataset
零售绩效与盈利能力诊断仪表盘

本项目基于SuperstoreSalesDataset，通过数据分析方法对零售业务的销售表现与盈利结构进行探索性分析，并构建数据可视化仪表盘。

数据集来源：
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting/versions/1/data

项目概述
本项目使用Python对数据进行基础清洗与预处理，并利用MySQL进行数据存储与中转，最终通过PowerBI构建数据分析仪表盘。

分析重点包括：
企业销售收入变化趋势
产品类别盈利结构
客户贡献度分布
产品销售效率

不同于简单的数据可视化，本项目更关注通过数据分析识别潜在业务风险、发现业务增长驱动因素，并揭示可能存在的结构性盈利问题。

技术实现
数据清洗：Python（Pandas）
数据存储：MySQL
数据可视化：PowerBI

关键分析发现(KeyFindings)
1.明显的季节性销售模式
    销售收入在第四季度（Q4）持续达到峰值，说明企业销售具有明显的节假日驱动需求特征。
2.科技类产品是主要增长引擎
    Technology（科技类）产品类别贡献了最高销售额且利润率较高，是推动公司收入增长的重要业务板块。
3.存在结构性盈利问题
    部分子类别（如Tables）虽然销售额较高，但利润为负，说明可能存在定价策略不合理或成本控制问题。
4.收入集中度风险
    少数头部客户贡献了较大比例的销售收入，企业收入结构存在一定的客户集中度风险。
5.部分高销售客户利润为负
    部分客户虽然贡献了较高的销售额，但订单整体利润为负，说明企业在部分交易中可能存在过度折扣或成本控制不足的问题。
